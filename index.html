Aidan's CSS framework


This css framework is a collection of CSS styles to make simple websites like this just a little bit nicer. Now you can write your simple static site with nice semantic html, and the framework will manage the styling for you.

Buttons

Buttons are styled to stand out and provide feedback, with distinct styles for different states and purposes: normal, success, error, and disabled.

Text Inputs

Text inputs are designed to be visually consistent and user-friendly, with padding and borders that align with the overall design language.

Links

Links are clearly distinguishable and feature hover effects to improve user experience and navigability within and across web pages.

Images

Images are automatically adjusted to fit their containers, maintaining aspect ratio without stretching, for a clean and responsive design.

Typography


Our framework defines styles for all standard HTML heading elements, ensuring a clear hierarchy and visual appeal in textual content.
